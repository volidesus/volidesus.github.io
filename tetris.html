Volidesus

Space Invader Pac-Man Pong Break-Out Tetris

CONTROLS

Left: A, Right: D,
Turn: W, Turn Down: S

HOW TO PLAY

Tetris is played by using the pieces to create as many complete
horizontal rows of blocks without empty space, but if the Tetriminos
surpass the Skyline the game is over!

Keep in mind this is an open source project
and can be accesed through github.com
